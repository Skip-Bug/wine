# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Запуск
Инструкция написана для запуска на Windows
Откройте командную строку (CMD) или PowerShell
   - Нажмите `Win + R`, введите `cmd` и нажмите Enter
Перейдите в папку с программой:
   ```bash
cd укажите_путь_к_папке_wine
   ```
Проверьте версию Python:
   ```bash
python --version
   ```
- Должна быть версия 3.8 или выше

Установите необходимые библиотеки:
   ```bash
pip install -r requirements.txt
  ```
Запустите программу:
   ```bash
python3 main.py
   ```
Работать с файлом данных по товару:
Первый способ:
   ```bash
python main.py -w путь_к_вашему_файлу.xlsx
   ```
Или измените данные в файле который стоит по умолчанию `wine3.xlsx`

Второй способ: возможность использовать ваш файл, если вместо `wine.xlsx`
поставите свой файл или полный путь до вашего файла
   ```bash   
set WINE_CATALOG=wine.xlsx && python main.py
   ```
Для powershell:
```powershell
 $env:WINE_CATALOG="путь_к_вашему_файлу.xlsx"; python main.py
```
Чтоб вернуть по умолчанию сделайте
   ```bash
set WINE_CATALOG=
   ```
Для powershell:
 ```powershell
$env:WINE_CATALOG=$null
```
Если все правильно сделали то в строке `Использую файл: будет ваш_файл.xlsx`
Проверте работу сайта:
Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).   
Убедитесь что все корректно отобразилось.   

Остановить работу программы:
Нажмите `Ctrl + C` в окне консоли или закройте консоль.
